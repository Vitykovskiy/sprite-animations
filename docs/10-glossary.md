## Технические термины

| Термин                 | Определение                                           |
|------------------------|-------------------------------------------------------|
| Epic                   | Крупный блок работы, объединяющий связанные задачи    |
| Task Intake            | Фаза качественного снятия бизнес-задачи               |
| Vector DB              | База данных для хранения векторных представлений данных |
| Embedding              | Векторное представление текста или другого контента   |
| Kanban                 | Метод визуального управления задачами через доску     |
| DoR                    | Definition of Ready — критерии готовности задачи к работе |
| DoD                    | Definition of Done — критерии завершённости задачи    |
| ADR                    | Architecture Decision Record — запись об архитектурном решении |
| Canonical source       | Единственный авторитетный источник правды по данным   |
