## Краткое описание проекта

<!-- Агент заполняет: 2-3 предложения о том, что это за проект -->
`sprite-animations` — библиотека для 2D sprite-анимации в вебе на `canvas`, ориентированная на внутреннее использование и полный контроль над runtime-поведением. Текущая версия включает npm-пакет, поддержку regular grid sprite sheet и frame sequence, а также dev playground на Vue 3 для настройки, проверки и сохранения конфигурации анимации.

---

## Цель проекта

<!-- Агент заполняет: конкретная, измеримая цель -->
Подготовить рабочий MVP библиотеки, которая умеет загружать регулярный sprite sheet и frame sequence, воспроизводить анимацию на `canvas`, управлять таймингом и позиционированием, а также поддерживает настройку параметров через playground.
